RETURN POLICY

Membership Tier Return Windows:
- Gold Tier: 30-day return window from delivery date
- Silver Tier: 20-day return window from delivery date
- Bronze Tier: 15-day return window from delivery date

Membership Tier Refund Percentage:
- Gold Tier: 100% refund of purchase price
- Silver Tier: 75% refund of purchase price
- Bronze Tier: Refund percentage determined by product condition assessment via vision agent

Conditions for Returns:
- Items must be unused and in original packaging
- All parts, accessories, and manuals must be included
- No physical damage, stains, odors, or signs of use
- Packaging must be intact and undamaged
- Electronics must be wiped of all personal data
- Serial numbers must match original purchase records

Return Process:
1. Log into your account and navigate to Order History
2. Select the item you wish to return
3. Choose your return reason
4. Print the prepaid return label
5. Repackage items securely in original packaging
6. Include all parts, accessories, and documentation
7. Drop off at any authorized shipping location

Refund Timeline:
- Refund processed within 5-7 business days after receiving returned item
- Original payment method will be credited
- Shipping costs are non-refundable unless item is defective or damaged in transit

Special Cases:
- Defective items: Full refund or replacement including shipping costs
- Damaged in transit: Full refund including shipping; photo evidence required
- Missing parts: Replacement parts shipped at no cost if available
- Items used with food: No returns accepted for hygiene reasons
- Custom or personalized items: No returns unless defective
